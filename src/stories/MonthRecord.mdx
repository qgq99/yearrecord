# MonthRecord
可展示某个月的数据记录情况, 每个组件都由7行不定列的小方块组成。 形式上, 位于第一行的方块代表当天是周一, 第二行代表周二, 
依次类推。最左边一列从上到下第一个显示的方块代表本月1号, 日期递增方向为做上到下, 从左到右。
每个小方块的颜色深浅表示了该天数据的大小, 数据越大, 其颜色越深。

## 数据
MonthRecord接收的数据类型为`DataRecord[] | undefined,`, 若为`undefined`, 则自动生成类型为`DataRecord[]`的数据。类型DataRecord的定义如下:
```js
interface DataRecord {
  year: number,
  month: Month,
  day: Day,
  data: number
}
```
*尽管不会报错, 仍建议data的长度恰好等于所指定的月份的天数, 其日期按真实顺序递增*

# 指定Tooltip title的内容计算逻辑
`tooltipTitileFunc`这个prop接收一个类型为`(item: DataRecord) => string`的函数, 改函数将接收每一条数据记录作为输入, 并将其返回值设置为其Tooltip title的内容。
例如, 当设置:
```
tooltipTitileFunc = item => `${item.year}年${item.month}月${item.day}日, 数据值: ${item.data}`
```
